# Step 1: ls コマンドを試してみよう

Linuxの基本コマンド `ls` を使って、ファイルやディレクトリの一覧を表示する方法を学びましょう！

---

## ls コマンドとは？

`ls` は、現在のディレクトリ（フォルダ）にあるファイルやディレクトリを一覧表示するコマンドです。  
オプションを付けることで、表示方法をカスタマイズできます。

---

## 課題①：基本の ls コマンド

まずは、シンプルに `ls` を実行して、ファイル一覧を表示してみましょう。

```bash
ls
```{{exec}}

確認：ファイルやディレクトリの名前（例：`test` や `test_dir`）が表示されることをチェックしてください。

---

## 課題②：隠しファイルを表示（-a オプション）

通常、`ls` では「.（ドット）」で始まる隠しファイルは表示されません。  
`-a` オプションを使って、隠しファイルも表示してみましょう。

```bash
ls -a
```{{exec}}

確認：`.hidden` や `.bashrc` などの隠しファイルが表示されることをチェック。

---

## 課題③：詳細表示（-l オプション）

`-l` オプションを使うと、ファイルの詳細情報（権限、所有者、サイズなど）が表示されます。

```bash
ls -l
```{{exec}}

確認：各ファイルの詳細情報（例：`drwxr-xr-x` やサイズ）が表示されることをチェック。

---

## ls コマンドの書式

```bash
ls [オプション] [ファイルやディレクトリ]

よく使うオプション
-a：隠しファイルも含めて全て表示（All）

-l：詳細情報を表示（Long形式）
